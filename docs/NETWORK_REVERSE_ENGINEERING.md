## üñ•Ô∏è Browser Network Capture Methods

### 1. Chrome DevTools Network Tab (Easiest)

1. **Open DevTools:** Press `F12` or `Ctrl+Shift+I`
2. **Go to Network Tab:** Click "Network" tab
3. **Filter by:**
   - `XHR` - API calls
   - `Fetch` - Modern API requests
   - `WS` - WebSocket connections
   - `Doc` - Page loads
4. **Preserve Log:** Check "Preserve log" to capture navigation
5. **Export HAR:** Right-click ‚Üí "Save all as HAR" for analysis

### 2. Chrome `chrome://net-export/` (Full Packet Capture)

Chrome has a built-in packet export feature:

1. **Navigate to:** `chrome://net-export/`
2. **Click "Start Logging to File"**
3. **Perform actions** on the target site (login, search, apply)
4. **Click "Stop Logging"**
5. **Open in Chrome UI:** Load the file in `chrome://net-internals/#import`

### 3. Firefox Network Logger

1. **Open DevTools:** `Ctrl+Shift+E`
2. **Filter:** XHR, Fetch, WS
3. **Persist:** Check "Persist Logs"
4. **Right-click ‚Üí "Save All As HAR"**

### 4. Using mitmproxy (Most Powerful)

For full HTTPS interception:

```bash
# Install mitmproxy
brew install mitmproxy

# Start proxy
mitmproxy

# Configure browser to use proxy
# Visit http://mitm.it to install certificate
```

### 5. Browser Console Commands

```javascript
// Monitor all fetch requests
const originalFetch = window.fetch;
window.fetch = (...args) => {
  console.log('FETCH:', args[0]);
  return originalFetch(...args).then((response) => {
    console.log('RESPONSE:', response.url, response.status);
    return response;
  });
};

// Monitor all XHR requests
const originalXHR = XMLHttpRequest.prototype.open;
XMLHttpRequest.prototype.open = function (method, url) {
  console.log('XHR:', method, url);
  return originalXHR.apply(this, arguments);
};

// Capture all API responses
document.addEventListener('DOMNodeInserted', (event) => {
  if (event.target.src) {
    console.log('LOADED:', event.target.src);
  }
});
```

## üîß Automation Patterns Discovered

### Sonara.ai Pattern:

1. **Upload Resume** ‚Üí Parse and store
2. **Set Preferences** ‚Üí Save to user profile
3. **Auto-Search** ‚Üí Polls job APIs every 15 minutes
4. **Match & Score** ‚Üí AI ranking algorithm
5. **Auto-Fill** ‚Üí Forms filled via automation scripts
6. **Submit to ATS** ‚Üí Direct integration with Workday/Greenhouse

### LazyApply.com Pattern:

1. **Chrome Extension** ‚Üí Intercepts LinkedIn/Indeed pages
2. **One-Click Apply** ‚Üí DOM manipulation to fill forms
3. **AI Generation** ‚Üí Calls OpenAI API for cover letters
4. **Analytics** ‚Üí Tracks all applications in dashboard

### JobCopilot.com Pattern:

1. **Weekly Subscription** ‚Üí $8.90/week model
2. **Human-like Behavior** ‚Üí Random delays, mouse movements
3. **Direct Company Sites** ‚Üí Avoids job boards, goes to source
4. **Review Mode** ‚Üí Users can edit before submission

### JavaScript Analysis:

```javascript
// Find all API calls on page
const apis = new Set();
document.querySelectorAll('[src], [href]').forEach((el) => {
  const url = el.src || el.href;
  if (url.includes('/api/') || url.includes('/v1/')) {
    apis.add(url);
  }
});
console.log('Found APIs:', Array.from(apis));
```

## üöÄ Using Captured Data for Implementation

Once you have the API patterns, implement similar features:

```typescript
// Example: Job search API implementation
class JobSearchService {
  async searchJobs(params: JobSearchParams): Promise<Job[]> {
    // Implement similar to competitor
    const response = await this.apiClient.get('/api/jobs/search', {
      params: {
        query: params.query,
        location: params.location,
        limit: params.limit || 50,
      },
    });

    return response.data.jobs.map((job) => ({
      id: job.id,
      title: job.title,
      company: job.company,
      url: job.url,
      matchScore: this.calculateMatchScore(job, params),
    }));
  }
}
```
