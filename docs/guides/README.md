### Key Concepts

#### Workflows

Automated sequences of steps that process data and perform actions.

- **Triggers**: Events that start workflows (webhooks, schedules, manual)
- **Nodes**: Individual steps in a workflow
- **Connections**: Links between nodes that pass data
- **Execution**: A single run of a workflow

#### Integrations

Connections to third-party services and applications.

- **Zapier MCP**: Pre-authenticated access to 30,000+ apps
- **n8n Nodes**: Custom workflow processing nodes
- **Webhooks**: Real-time event notifications
- **API**: Direct API access for custom integrations

#### AI Models

Free AI models powered by Puter.js.

- **gpt-3.5-turbo-free**: General conversation and text generation
- **code-davinci-free**: Code generation and debugging
- **text-davinci-free-002**: Text completion and summarization
